# Week 2 - Day 1: JavaScript Foundation & DOM

## Core Concepts Learned

### 1. JavaScript Fundamentals

- **Purpose:** Provides logic, interactivity, and dynamic behavior to web pages.
- **Integration:** Best practice is connecting an external file using `<script src="script.js"></script>` just before the closing `</body>` tag.

### 2. Variables & Data Types

- **`let` vs `const`:** Use `let` for values that will be reassigned; use `const` for immutable values. Avoid `var`.
- **Data Types:** Handled core types including `String` (text), `Number` (integers/decimals), `Boolean` (true/false), and `Object` (key-value collections).

### 3. Operators

- Utilized assignment (`=`), mathematical (`+`, `-`), and strict comparison (`===`) operators to evaluate expressions.

### 4. The DOM (Document Object Model)

- **Definition:** The browser's internal, object-oriented representation of the HTML structure. It serves as the API that allows JavaScript to interact with and modify the page layout and content.

## Project

Created an HTML document linked to an external JavaScript file. Utilized `console.log()` to output various data types, perform basic math operations, and inspect the `document` object to visualize the DOM tree within the browser's Developer Tools console.
