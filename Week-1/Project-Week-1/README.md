# Week 1 Complete: Frontend Foundations 🚀

## Overview
This week established the core building blocks of the web: HTML (structure) and CSS (styling and layout). We progressed from basic tags to building a complete, responsive, and semantic web page.

## Master Concept Checklist

### 1. Document Structure & Semantics
* **Boilerplate:** `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`.
* **Semantic HTML:** Utilizing tags like `<header>`, `<main>`, `<section>`, `<nav>`, and `<footer>` instead of meaningless `<div>` tags to improve SEO and code readability.

### 2. Core HTML Elements
* **Text Formatting:** Headings (`<h1>` to `<h6>`) and Paragraphs (`<p>`).
* **Media & Links:** Anchor tags (`<a>` with `href` and `target`) and Images (`<img>` with `src` and `alt`).
* **Data Structuring:** * Lists (`<ul>`, `<ol>`, `<li>`).
    * Tables (`<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, `<td>`).

### 3. User Input (Forms)
* **Structure:** The `<form>` container.
* **Elements:** Labels (`<label>`) attached to Inputs (`<input type="...">`), Dropdowns (`<select>`/`<option>`), and submit buttons.

### 4. CSS Styling Basics
* **Integration:** External stylesheets linked via `<link rel="stylesheet" href="...">`.
* **Typography:** System fonts vs. external Web Fonts (Google Fonts via `<link>`).
* **Colors:** Utilizing Hex Codes (e.g., `#FFFFFF`), RGB/RGBA (e.g., `rgba(0,0,0,0.1)`), and HSL (e.g., `hsl(210, 20%, 98%)`).

### 5. Layout & Architecture
* **The Box Model:** The critical understanding that every element has Content, Padding (inner space), Borders, and Margins (outer space).
* **Flexbox:** Applied via `display: flex;` to align and space children dynamically using `justify-content` (main axis), `align-items` (cross axis), and `flex-direction` (row vs. column).

## Capstone Application
Engineered a complete, responsive SaaS landing page incorporating a sticky navigation bar, a multi-column hero section, feature lists, a data table for pricing, and a styled waitlist form, acting as a culmination of all Week 1 concepts.
